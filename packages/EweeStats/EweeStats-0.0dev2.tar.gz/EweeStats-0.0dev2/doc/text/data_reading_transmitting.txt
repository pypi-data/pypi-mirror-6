
Lecture et transmission des mesures
***********************************

Pour lire les capteurs, ils ont été branchés à l'Arduino. La
transmission se fait par port USB, ce qui permet d'alimenter l'Arduino
et de transférer les données rapidement, et assez facilement.


Transmission côté Arduino
=========================

Il était nécessaire de pouvoir accéder aux valeurs reçues par
l'Arduino à n'importe quel moment. C'est pourquoi le programme Firmata
a été utilisé. Une fois envoyé sur l'Arduino, celle-ci peut envoyer
tout ce qu'elle reçoit par le port USB. Le programme est disponible en
tant qu'exemple dans l'IDE Arduino, nommé StandardFirmata. Sinon, il
est téléchargeable sur le site Firmata.


Transmission côté Raspberry Pi
==============================

Une fois que Firmata est sur l'Arduino, il faut utiliser une librairie
python pour communiquer avec l'Arduino. pyFirmata a été utilisée pour
cela.
