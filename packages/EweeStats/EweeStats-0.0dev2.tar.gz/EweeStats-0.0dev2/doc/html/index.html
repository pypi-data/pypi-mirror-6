Documentation de EweeStats¶

EweeStats est un programme d’analyse et de traitement des données destiné à l’Ewee. Il lit les données en analogique, les traite et les présente. EweeStats démarre au lancement de la RPi et enregistre toutes les données tant que l’Ewee fonctionne.
