
Générisation des opérations
***************************

Afin de faciliter l'utilisation du programme, il faut génériser cette
opération, c'est-à-dire l'adapter à une variable. Ce qui varie en
fonction des utilisations ici, c'est le nombre de capteurs. C'est
pourquoi la variable analogSensors contenant un entier naturel est
globale.

Afin d'adapter les opérations au nombre de capteurs, les boucles for
ont été utilisées. Par exemple :

           start = time.time()
           while board.analog[0].read() is None:

builtins.range(stop) -> range object

   range(start, stop[, step]) -> range object

   Return a virtual sequence of numbers from start to stop by step.
