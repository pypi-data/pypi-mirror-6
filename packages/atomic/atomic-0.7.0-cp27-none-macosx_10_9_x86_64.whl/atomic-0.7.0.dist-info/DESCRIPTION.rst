======
Atomic
======

An atomic class that guarantees atomic updates to its contained value. ::

    from atomic import Atomic
    atomic = Atomic(0)
    atomic.update(lambda v: v + 1)


Installation
============

To install atomic, use pip : ::

    pip install atomic
